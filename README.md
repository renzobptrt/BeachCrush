# Beach Crush
Same to candy crush. Compiled in Unity 2019.3.1f1 version

## Features
- Type: 2D
- Point of view: Third person
- Gender: Puzle
- Platform: Mobile
- Graphics engine: Unity 2D
- Main language: C#
- Level : 1
